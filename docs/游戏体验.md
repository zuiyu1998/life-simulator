- 解谜

# 主要流程

总结版: 玩家通过劳动实现自己的梦想。

玩家在不同的人生阶段（商店）会遭遇不同的事件。
解决不同的事件会获得不同的收获，比如获得更强的劳动，金钱，人生阶段中事件的刷新机制等。
不解决事件只会获得被动增益，由环境决定。
玩家需要通过事件提升自己的劳动能力，并在最终挑战到来时，实现终极挑战。
死亡和完成挑战代表游戏结束，会解锁人生评语。

成长: 技能的成长，金钱的成长，心态的成长，环境的成长，身体的成长等
解谜: 每个事件有不同的解法。

事件挑战(关卡):
在体力，心情，金钱的限制下通过劳动解决事件。
劳动有收获，但同时也会消耗。
短期：玩家需要避免自己的状态归零，
长期：需要提升的能力值。
好的方案：平衡长期和短期的数值。
